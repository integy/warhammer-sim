# ⚔️ Warhammer 40K WTC Simulator

Web-based Warhammer 40K tabletop simulator following WTC (World Team Championship) rules.

## 🌐 Live Demo

https://integy.github.io/warhammer-sim/

## 🎮 Features

### WTC Rules
- 5 Rounds, 3 Hours total
- Win: 20BP | Draw: 15BP | Loss: 12BP
- Standard WTC scoring

### Phase System
1. Command Phase
2. Movement Phase
3. Shooting Phase
4. Charge Phase
5. Fight Phase
6. Morale Phase

### Dice Roller
- Hit Rolls (BS/WS)
- Wound Rolls (Strength vs Toughness)
- Save Rolls (AP modified)
- Morale Tests
- Reroll support (1s, failed)
- Modifiers (+/-)

### Army Builder
- Add custom units
- Point tracking
- Unit stats (WS, BS, S, T, W, LD, SV)
- Weapon profiles
- Save/Load army (localStorage)

### Game Board
- Visual battlefield with grid
- Deployment zones
- 5 Objective markers
- Turn/Round tracking

### Score Tracking
- Primary Objectives
- Secondary Objectives
- Total Battle Points

## 🚀 Usage

Simply open `index.html` in any modern browser - no server required!

Or serve locally:
```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080
